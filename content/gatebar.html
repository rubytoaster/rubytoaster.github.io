To Get Started:

Create a name for your chart. Define the number of gates for your process machine. Establish the expected requirement that will apply to each gate.

For each gate, you will be prompted to provide a name for the gate. Enter the remaining days, actual days and last average number of days for each gate.

Continue until all gate information has been entered. Then create the chart.
