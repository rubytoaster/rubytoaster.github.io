New Table Chart
To Get Started:

Create a name for your chart. Define the available days for your project and the required output to complete that number of days.

Establish either the WIP goal or the Flowdays goal. Entering one will calculate the other.

Determine the number of gates. For each gate, you will be prompted to provide a name for the gate and either the WIP or Flowdays. Entering one will calculate the other.

Continue until all gate information has been entered. Then create the chart.
